About Web Desk

Use this page as a quick guide for Web Desk tools, buttons, and pop-up menus.

Side Bar Icon Pop-up Other Buttons Export / Import

Left Side Bar Overview

Everything you need to build, manage, and customize your dashboard.

  1. Background Change the background color.
  2. Switch Mode Toggle between Edit Mode and Active Mode. Edit Mode: Drag icons to arrange your dashboard. Active Mode: Click an icon to open its linked website.
  3. Delete Remove selected icons.
  4. Add Icon Create a new icon (see next section for Icon creation menu).
  5. Load Load a saved layout.
  6. Save Save your current layout.
  7. Clear Board Remove all icons.
  8. Grid Snap Align icons to a grid.

Icon Pop-up Overview

This menu is used to create a new icon for your dashboard.

  1. Icon TypeChoose between a shape icon or an uploaded image.
  2. Shape PreviewShows what the shape will look like before creating it.
  3. ShapeSelect the shape used for the icon.
  4. Shape ColorChange the color of the icon shape.
  5. Display LabelSet the text shown under the icon.
  6. Text ColorChange the label text color.
  7. LinkAdd the website the icon should open.
  8. Create / CancelCreate the icon or close the pop-up without saving.
  1. Image PreviewShows what the image will look like before creating it. The upload is converted to 128 × 128 pixels while keeping its image ratio.
  2. Image SelectionOpens your file window to select your image.

Other Buttons Overview

These buttons help with saving layouts, importing data, and switching window modes.

  1. ExportCopies your saved Web Desk data so you can back it up or move it to another browser.
  2. ImportLets you paste previously exported Web Desk data back into the app.
  3. FullscreenExpands the dashboard area and hides the top bar and sidebar.
  4. Exit Full Screen, currently in edit modeExit Expanded Dashboard, returning top and side bar, showing current mode while in fullScreen as Edit Mode.
  5. Exit Full Screen, currently in active modeExit Expanded Dashboard, returning top and side bar, showing current mode while in fullScreen as Active Mode.
  6. Hover Tips ToggleGives a quicktip when hovering over icons in Active or Edit mode
  7. Icons These are your custom created icons displayed on the dashboard. Edit Mode: Drag icons and click to edit them. Active Mode: Click to open the link.

Export & Import Layout

Save your dashboard or load a previously saved layout.

  1. Export DataDisplays your current dashboard data as text that can be copied.
  2. Cancel ExportCloses the export window without copying the data.
  3. CopyCopies the export data to your clipboard.
  4. Import Data AreaPaste your previously exported dashboard data here.
  5. Cancel ImportCloses the import window without applying changes.
  6. Commit ImportApplies the pasted data and loads the saved dashboard.
